# Easter Celebration Page

A simple, inspirational webpage celebrating Easter — featuring an animated dawn/sunrise effect, golden light rays, floating particles, scripture verses, and a message of hope and renewal.

## View

Open `index.html` in any modern browser.

## Features

- Animated sunrise hero with glowing cross silhouette
- Floating golden light particles
- Scroll-triggered reveal animations
- Inspirational message and scripture cards
- Fully responsive design
- Pure HTML/CSS/JS — no dependencies
